Parent Component how to pass data from parent to child Use @Input decorators to share data from Parent to Child component.

@Input() decorator in a child component or directive to let Angular know that a property in that component can receive its value from its parent component. It helps to remember that the data flow is from the perspective of the child component. So an @Input() allows data to be input into the child component from the parent component.
