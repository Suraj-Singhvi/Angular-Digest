#1. Using ng-template elements in Angular 2 Content of templates are not rendered on its own directly
Content written in ng-template are rendered as it is ...they are not wrapped in ng-template tag in DOM.
to show content of ng-template we need to use a template ref & then use ViewChild to query and get reference to the template & then embed the template using ViewContainerRef
